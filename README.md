## 🙋‍♀️ 스터디 진행 방식
시간 : 매주 목요일 저녁 7시
장소 : 홍대
공부 방식 : 팀원마다 여러 주제 중 하나를 골라 그 주제를 중심으로 설명. 공부는 각자의 스타일대로 ( 인터넷 강의, 구글링, 책 등 )
설명 방식 : 주제의 키워드에 대해 공부한 걸 이야기하면서 / 주제와 관련된 본인의 경험을 이야기하면서
기록 방식 : 본 레포지토리에 PR로 기록
인증 방식 : PR 리뷰
